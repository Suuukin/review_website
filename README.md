# Game Review Site

A personal game review website where I write about the games I've been playing. Each review is paired with info pulled directly from Steam so you can see the game's trailer, screenshots, and store page right alongside my thoughts.

## What the Site Does

- **Read Reviews** — Browse the full collection of reviews on the homepage, sorted newest first. Each entry shows the game's cover art, title, genres, tags, and the date I played it.

- **Search & Discover** — Search by game title, tag, or genre. Results show matching text highlighted so you can see exactly why something turned up. The sidebar panels on desktop pages set the mood with the game's own Steam background art.

- **Steam-Powered Game Pages** — Reviews for Steam games get a dedicated layout with the game's header image, a direct link to the Steam store, and the full Steam description baked right into the page.

- **Tags with Color** — Every review can have color-coded tags (e.g., "Must Play", "Skippable", "Hidden Gem") so you can quickly spot patterns across the collection.

- **Blog Posts Too** — Alongside game reviews, there's support for plain blog-style posts for anything that doesn't fit the game review format.

- **Admin Features** — I can create, edit, and delete reviews through a password-protected area. The editor supports full Markdown with a live preview, and there's an autocomplete game search that hooks into Steam's catalogue so I never have to type an app ID manually.

## Who It's For

Anyone who wants to read honest, no-nonsense opinions about games — mostly indie and Steam Next Fest demos, with occasional full-release coverage. I call out what works, what doesn't, and whether something's worth your time.
